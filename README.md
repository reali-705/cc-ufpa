# Análise de Algoritmos - Atividade 01

## Descrição
Este projeto contém implementações e análises dos algoritmos como:
- Insertion Sort usando busca linear
- Insertion Sort usando busca binária
- Heap Sort recursivo
- Heap Sort iterativo
- ABB Sort
- AVL Sort


## Conteúdo
- Implementação de algoritmos estudados
- Análises de complexidade de tempo
- Comparação de desempenho entre diferentes abordagens

## Como Executar
1. No main.py, escolha qual teste queria realizar.
2. Execute os scripts conforme necessário

## Objetivos
- Compreender a análise de complexidade algorítmica
- Implementar soluções eficientes
- Comparar diferentes estratégias de solução

## Requisitos
- Python 3.8+
- Pandas
