# TRABALHO DE PROJETO DE ALGORITMOS I
Esse repositório separa as atividades de códigos em diretórios.  
As atividades buscam por em prática o que foi ministrado em sala de aula através de códigos JavaScript e HTML.
## ATIVIDADE 1
Criando uma página com HTML e JavaScript.  
Adicionando uma foto e uma descrição do jogo escolhido.
## ATIVIDADE 2
Utilizando variáveis, funções, objetos e classe que estão conectadas com o jogo Angrys Birds.  
A página HTML contem a descrição dos elementos utilizados no JS e mostra o teste de 10 lançamentos.
## ATIVIDADE 3
Criação de um diagrama para elaborar classes relacionadas ao jogo. Para a criação e visualização do diagrama, utilizei uma extensão do vscode: **PlantUML**.  
Escrever um módulo que contenha as classes e importar no código principal.
